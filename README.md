# my_sh
.zsh &amp; .bashrc &amp; terminal related configs

## Install

### Git clone

```bash
cd ~
git clone git@github.com:MarMun/my_sh.git .my_sh
```

### Copy / adjust RC-Examples

```bash
cp .bashrc.example .bashrc
cp .zshrc.example .zshrc
```

### Embed in local configs

```bash
cd .my_sh
./install.sh
